LiveOak Secured Chat HTML

More info

The HTML Secured Chat is a perfect application to try out LiveOak server and start exploring its capabilities.

  • A user with the 'admin' role can do anything (subscribe, create new chat messages, view all messages received from subscription, delete any message).
  • A user with the 'user' role can view existing messages, create new messages and subscribe to receive messages. User with this role can delete just their own messages.
  • A user that doesn't have any roles can't do anything (view existing messages, create new messages, subscribe to receive messages).

You should see the sent text display in both windows instantly.
